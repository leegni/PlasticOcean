HOW TO USE:
Put "VRHandsDrawer" script on the main camera.
Put your's meshes to the "Hands List" in a script attached to camera.
You can ajust inside colors in "VRHandsInternalRim" material.
For properly look, materials of yours hands need to use shaders that provided with this pack: "VRHands/Rim", "VRHands/Unlit", "VRHands/Bumped Specular", "VRHands/Standard".
